## Product Goal

Build a SaaS platform for KYC, AML, identity verification, watchlist screening, risk scoring, case management, and auditability for fintech operators.

## MVP Modules

- Identity verification
- Document OCR
- Watchlist screening
- Decision engine
- Case management
- Audit trail
- API platform

## Important Constraints

- Multi-tenant
- Secure by default
- API-first
- Modular
- Supabase-backed
- Next.js dashboard on App Router
